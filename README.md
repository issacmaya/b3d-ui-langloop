## 正體中文
### 外掛目的
本外掛是給 非英語系使用者使用，在學習 Blender 時，常常接觸其他語系的教學，這時候非英語系的使用者，可以透過本外掛 快速切換 增加學習效率
### 使用
- 安裝後，請用 ctrl alt shift T 切換語系
- 编辑 > 偏好设定 > 附加元件 > 搜索 'b3d-ui-langloop' 中設定
  - 循環數量
  - 語系欄位 ( 最多 10 組 )
### 設定
如果要修改快速鍵 :
编辑 > 偏好设定 > 快捷键 > 搜索 'b3d-ui-langloop'

## 日本語

このアドオンは非英語圏ユーザー向けです。Blender学習中に他言語のチュートリアルに触れるとき、素早く言語を切り替えて学習効率を上げます。

  - 言語フィールド (最大10組)

ショートカットを変更する場合：
編集 > プリファレンス > キー設定 > 'b3d-ui-langloop' を検索

## English
### Addon Purpose
This addon is for non-English speakers. When learning Blender with tutorials in other languages, quickly switch to improve efficiency.
### Usage
- After installation, Ctrl Alt Shift T to switch languages
- Edit > Preferences > Add-ons > Search 'b3d-ui-langloop' to set
  - Cycle count
  - Language fields (up to 10 groups)
### Settings
To modify shortcut:
Edit > Preferences > Keymap > Search 'b3d-ui-langloop'
