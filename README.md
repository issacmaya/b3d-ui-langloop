## 正體中文
### 外掛目的
本外掛是給 非英語系使用者使用，在學習 Blender 時，常常接觸其他語系的教學，這時候非英語系的使用者，可以透過本外掛 快速切換 增加學習效率
### 使用
安裝後，請用 ctrl alt shift T 切換語系
內建 10組 語系欄位
### 設定
如果要修改快速鍵 :  
编辑 > 偏好设定 > 快捷键 > 搜索 'b3d-ui-langloop'
